### github 使用步驟
-專案
-git init
-git add .
-git commit -am "init"
-github 專案網址
-git add remote remote "github專案網址"
-git push -u origin master

-回家
-git clone 專案網址
-編修檔案
-git add .
-git commit -am "家裡修正版本"
-git push
-關機

-git pull(雲端到本地端)
